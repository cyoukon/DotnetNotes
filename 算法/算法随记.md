1. 判断一个<font color='red'>**正**</font>整数是不是2的n次方

   ```C#
   (num & (num - 1)) == 0 ? "2的n次方" : "不是2的n次方"
   /*
    * 100 & 011      4 & 3
    * 1000 & 0111    8 & 7
    */
   ```

2. 不使用临时变量交换两变量值

   ```C#
   // 对所有类型
   string a = "a";
   string b = "b";
   (b,a) = (a, b);
   ```

   ```C#
   // 对int类型
   int a = 1;
   int b = 2;
   // 加减计算法
   a = b - a;
   b = b - a;
   a = b + a;
   // 异或运算法
   a = a ^ b;
   b = a ^ b;
   a = a ^ b;
   // 变量本身交换数值
   b = (a + b) - (a = b);
   ```
